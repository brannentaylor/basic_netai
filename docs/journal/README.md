# Build journal

Each entry is a dated markdown file (example `2026-05-09-topic.md`) with:

1. **Context**
2. **Goal**
3. **Prompt** (when AI-assisted)
4. **Answer / outcome**
5. **Repo changes** (commit SHA)
6. **Problems + resolution**
7. **Share hook** (one-liner for social)

Link related ADRs under `docs/decisions/`.
